The Two Most Confusing methods of an array in Javascrip

Hey everyone, in this blog I am going to discuss the 2 array method. These methods are most confusing for most people.

What is Array?

An array is a collection of data. We use arrays to store multiple types of data in JS array.

var arrayList = [1, 3, 5 ,”Milk”, “bread”, 3.12 ,true];

likewise, we can store integer, string, Boolean, decimals, etc.Let’s start with the first method.

slice() :

Slice() method returns a copy of a portion in an array into a new array, selected from start to end. As a new array object, It returns the selected elements in an array. slice method takes two arguments like slice(start, end).The method selects an element from the start argument and up to end argument but end argument is not included.

Following are examples :

var array = [1,2,3,4,5,6];

var newarray = array.slice(3);

slice method slice the first 3 elements of original array and gives a new array object as newarray = [4,5,6];

another example is using start and end argument . var array = [1,2,3,4,5,6]; var newarray1 = array.slice(1,3); It gives new array object as newarray1 =[2,3].

splice():

splice method changes an array by adding or removing elements. It takes a 3 arguments, first argument defines a position where the new argument is added. The second argument defines how many elements will be deleted. The third argument defines a new element to be added.
